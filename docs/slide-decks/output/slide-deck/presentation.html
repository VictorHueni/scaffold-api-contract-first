Before we start
This demo is built on a classic three-tier application, not a complex enterprise ecosystem. The goal is to show what's possible with a contract-first approach, using simple, concrete examples.

After the demo we discuss what makes sense for our context and build something meaningful and useful for the company together.

Contract-First Maturity Model

L4 Mastery
L3 Scale
L2 Automate
L1
Mocks
L1 Foundation
Shared Mock Server
Centralized Prism instance fed from spec examples. Frontend, ESB, and QA share the same mock. No more diverging stubs.
Prism Prism docs
AI Spec
L1 Foundation
AI-Assisted Specification
Claude Code scaffold with business context and contract-first tools injected. Devs and BAs design specs with AI guidance, linting, and best practices built in.
Claude Code Spectral
Auth
L1 Foundation
Seamless Auth
Wire existing Vault credentials into Scalar try-it-out, Hurl tests, and Schemathesis runs. No more hunting for tokens on Slack.
Vault Scalar Auth
ZAP Scan
L2 Automate
Dynamic Security Scanning
OWASP ZAP runs active attacks against the mock or running API. Catches injection, auth bypass, and OWASP API Top 10 vulnerabilities at runtime. Complements the static Spectral OWASP ruleset already in the pipeline.
ZAP ZAP Docker ZAP Automation
Observability
L2 Automate
API Observability
Pipe API gateway logs and ESB monitoring events into the existing ELK stack. Build Kibana dashboards for call volume, latency percentiles, error rates, and deprecation signals per endpoint.
Kibana Filebeat
AsyncAPI
L3 Scale
AsyncAPI Events
Apply the same contract-first philosophy to event-driven APIs: Kafka topics, message queues, MQTT. Lint with Spectral, generate docs, and produce typed consumers from AsyncAPI specs.
AsyncAPI Generator Studio
Registry
L3 Scale
Schema Registry
Central versioned catalog for all contracts: OpenAPI, AsyncAPI, GraphQL, Protobuf. Discover existing schemas, detect duplicates, and govern shared definitions across teams.
Apicurio Backstage
SDK Gen
L3 Scale
SDK Generation
Auto-generate typed client SDKs (TypeScript, Java) from the spec and publish to internal registries. Requires consumer coordination and a distribution pipeline.
OpenAPI Generator Kiota
Prod Data
L3 Scale
Prod Data as Fixtures
Capture real production payloads, classify and strip PII, then feed anonymized data back as spec examples and test fixtures. Realistic tests, zero compliance risk.
Tonic.ai ARX Anonymizer
Full CI/CD
L4 Mastery
Full CI/CD Pipeline
End-to-end automation: PR triggers lint, mock, contract tests, artifact build, deploy, smoke test, and promote to production. No human in the loop from merge to prod.
GitHub Actions Tekton
Mapping IDE
L4 Mastery
Visual Mapping IDE
Vendor-neutral OSS tool to import any contract (OpenAPI, WSDL, AsyncAPI), visually link fields, define transformations, and generate output for any runtime: ESB maps, DataWeave, XSLT, and more.
OSS Project (WIP)
Traffic Val.
L4 Mastery
Traffic Validation
Proxy or sidecar captures live traffic in non-prod and validates against the spec. Catches contract drift that tests miss. Less critical with solid CI, but ultimate safety net at scale.

THE VISION

Imagine This

Today
A new developer joins. They ask on Slack:
"Where's the API doc? How do I get a token?" Three people answer. None agree.
3 months from now
Same developer joins. Opens the spec. Clicks "try it out."
Mocks respond. Tests pass. First PR by end of day. Nobody had to answer anything.
1 year from now
A breaking change lands. The pipeline catches it. The SDK rebuilds.
Every API on one dashboard. A PO browses the registry and scopes the next feature alone.
Production never flinches.
3 years from now
Every contract is discoverable. Every integration is tested.
Every deployment is automatic. Every mapping is visual.
The system runs itself. You design what matters.
